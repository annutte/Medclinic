# Medclinic
Сайт с ИИ-консультантом для клиники. Ассистент консультирует по услугам, стоимости и расписанию врачей и записывает на приём.

## 🌐 Сайт для клиники на Tilda 

- Страницы: Главная, Услуги (3), Расписание, Прайс-лист, Политика конфиденциальности, 404 
- Блоки: отзывы, контакты, форма обратной связи, поиск по сайту.  
- Виджет чата с ИИ-консультантом. 

---

## 🤖 ИИ-консультант  
- Отвечает на типовые вопросы.  
- Предоставляет доступные слоты для записи.  
- Собирает параметры записи (специализация/врач, дата, время).  
- Собирает контакты пациента (имя, телефон, email, дата рождения) в соответствии с требованиями 152-ФЗ.  
- Подтверждает запись и записывает её в базу данных.  
- Отправляет оповещение администратору.  
- Напоминает о приёме за день.
- Удаляет старые записи из базы данных.  
  
---

## ✏ Реализация
Помимо базового функционала реализован сложный алгоритм предоставления доступных слотов для записи без встроенных календарей:
- Нейросеть находит нужного специалиста по частичному совпадению: только по фамилии в любом падеже, по имени-отчеству или по однозначному соответствию (например, если в клинике только один невролог, то это однозначно <ФИО>).  
- Программным методом вычисляются все доступные интервалы у нужного специалиста с нужной даты на нужное количество дней (строится матрица слотов в соответствии с расписанием, из неё вычитается занятые слоты из таблицы записей).
